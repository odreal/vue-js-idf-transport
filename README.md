# projet-vue
## Instruction d'installation

## 1 - Installer les dépendances du projet :
```
npm install
```

### 2 - Tester le projet en mode développement :
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your end-to-end tests
```
npm run test:e2e
```

### Run your unit tests
```
npm run test:unit
```
